# Thai Number Reader

โปรเจกต์นี้เป็นโปรแกรมสำหรับแปลงตัวเลข (int) ให้เป็นคำอ่านภาษาไทย  
เช่น:

- 1 → หนึ่ง  
- 11 → สิบเอ็ด  
- 21 → ยี่สิบเอ็ด  
- 101 → หนึ่งร้อยหนึ่ง
